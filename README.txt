实验内容2
• 设计并实现一个LL(1)语法分析器，实现对算术文法E->E+T|T T->T*F|F F->(E)|i 所
定义的符号串进行识别，例如符号串98+99+80为文法所定义的句子，符号串(106-80(*95)不是文法所
定义的句子。
1. 检测左递归，如果有则进行消除；
2. 求解FIRST集和FOLLOW集；
3. 构建LL(1)分析表；
4. 构建LL分析程序，对于用户输入的句子，能够利用所构造的分析程序进行分析，并显示出分析过程。
